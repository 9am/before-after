## Usage

#### Install

```sh
npm install @9am/before-after 
```

#### HTML

```html
<script src="xxxxxxx/index.min.js"></script>

<before-after>
    <section slot="before">
        <h1>BEFORE</h1>
    </section>
    <section slot="after">
        <h1>AFTER</h1>
    </section>
</before-after>
```
